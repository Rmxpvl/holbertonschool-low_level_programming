# More Functions and Nested Loops

This project extends the work on functions and nested loops with more complex exercises including character classification, arithmetic, and drawing geometric shapes in the terminal.

## Files

| File | Description |
|------|-------------|
| `main.h` | Header file with all function prototypes |
| `0-isupper.c` | Function that checks if a character is uppercase |
| `1-isdigit.c` | Function that checks if a character is a digit |
| `2-mul.c` | Function that multiplies two integers |
| `3-print_numbers.c` | Function that prints the numbers from 0 to 9 |
| `4-print_most_numbers.c` | Function that prints the numbers from 0 to 9 except 2 and 4 |
| `5-more_numbers.c` | Function that prints numbers from 0 to 14, ten times |
| `6-print_line.c` | Function that draws a straight line using `_` |
| `7-print_diagonal.c` | Function that draws a diagonal line using `\` |
| `8-print_square.c` | Function that prints a square of `#` characters |
| `9-fizz_buzz.c` | Program that prints numbers from 1 to 100 applying the FizzBuzz rules |
| `10-print_triangle.c` | Function that prints a right-aligned triangle of `#` |

## Concepts Covered

- Character classification functions (`isupper`, `isdigit`)
- Integer arithmetic (multiplication)
- Printing patterns with nested loops (lines, diagonals, squares, triangles)
- FizzBuzz algorithm
- Writing modular, reusable functions with a shared header file

## Requirements

- All C files are compiled with `gcc -Wall -Werror -Wextra -pedantic`
- Programs are compiled on Ubuntu 20.04 LTS
- Functions use the prototype declared in `main.h`
