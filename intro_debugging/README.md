# Intro to Debugging

This project introduces the concept of debugging C programs by identifying and fixing logic errors in existing code. The programs cover basic mathematical functions and loop constructs.

## Files

| File | Description |
|------|-------------|
| `multiplication_table.c` | Prints a multiplication table of a given size using nested loops |
| `row_sums.c` | Computes the sum of each row and the total sum of a multiplication-style grid |
| `sum_to_n.c` | Computes the sum of all integers from 1 to `n` |

## Concepts Covered

- Reading and understanding existing C code
- Identifying off-by-one errors and incorrect loop conditions
- Tracing variable state through nested loops
- Debugging with `printf` statements and logical reasoning
- Writing correct accumulator patterns

## Requirements

- All C files are compiled with `gcc -Wall -Werror -Wextra -pedantic`
- Programs are compiled on Ubuntu 20.04 LTS
