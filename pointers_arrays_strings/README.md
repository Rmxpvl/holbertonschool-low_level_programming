# Pointers, Arrays, and Strings

This project covers the use of pointers, arrays, and string manipulation in C. It includes re-implementing standard library functions using only pointers and basic arithmetic.

## Files

| File | Description |
|------|-------------|
| `main.h` | Header file with all function prototypes |
| `0-reset_to_98.c` | Function that takes a pointer to an integer and updates its value to 98 |
| `1-swap.c` | Function that swaps the values of two integers using pointers |
| `2-strlen.c` | Function that returns the length of a string (`_strlen`) |
| `3-puts.c` | Function that prints a string followed by a newline (`_puts`) |
| `4-print_rev.c` | Function that prints a string in reverse |
| `5-rev_string.c` | Function that reverses a string in place |
| `6-puts2.c` | Function that prints every other character of a string |
| `7-puts_half.c` | Function that prints the second half of a string |
| `8-print_array.c` | Function that prints `n` elements of an integer array |
| `9-strcpy.c` | Function that copies a string from source to destination (`_strcpy`) |
| `0-strcat.c` | Function that concatenates two strings (`_strcat`) |
| `100-atoi.c` | Function that converts a string to an integer (`_atoi`) |

## Concepts Covered

- Pointers and pointer arithmetic
- Pass-by-pointer to modify variables from a function
- Arrays and their relationship to pointers
- String traversal and manipulation (reverse, copy, concat, half-print)
- Re-implementing standard C library functions: `strlen`, `puts`, `strcpy`, `strcat`, `atoi`

## Requirements

- All C files are compiled with `gcc -Wall -Werror -Wextra -pedantic`
- Programs are compiled on Ubuntu 20.04 LTS
- No standard library string functions used unless specified
